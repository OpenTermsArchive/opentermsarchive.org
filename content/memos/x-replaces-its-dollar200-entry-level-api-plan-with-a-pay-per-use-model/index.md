X [introduced](https://github.com/OpenTermsArchive/vlopses-us-versions/commit/18615aa) a new pricing model for its API called the "self-serve use". It is intended for "hobbyists, commercial prototyping, initial development, early-stage X product integrations, and supporting applications with a limited number of end-users", developers can now read and write content to the service and pay accordingly. Previously, it was a flat-rate system with only two options: a first one at $200 and a second one at $5,000, and restrictive rate-limiting. Commercial uses still have to subscribe to the pre-existing "Enterprise plan".
