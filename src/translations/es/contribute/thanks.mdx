## Ahora eres un colaborador del proyecto Open Terms Archive

Se utiliza el administrador de propuestas en GitHub para gestionar las contribuciones fácilmente.  Hemos creado un informe de problemas que <a href={url} target="_blank" rel="noopener">puedes ver aquí</a>.  Eso le permite mantenerse actualizado y aun así comunicarse con nosotros al respecto.

Hacemos todo lo posible para añadir su solicitud lo antes posible, pero puede tardar varios días. De hecho, para garantizar una alta calidad de los datos comprobamos cada contribución para asegurarnos de que todo funciona perfectamente.

Si quieres que te añademos a la lista de colaboradores que se publica en la página web OTA, simplemente envíenos un <a href="mailto:contact@opentermsarchive.org">correo electrónico</a> o un mensaje directo en nuestra <a href="https://twitter.com/OpenTerms" rel="noopener">cuenta de Twitter</a>.
