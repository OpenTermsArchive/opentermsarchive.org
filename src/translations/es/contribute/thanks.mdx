## Queremos que te sientas bienvenido en nuestro equipo

Con el propósito de gestionar las contribuciones fácilmente, utilizamos el administrador de propuestas en GitHub. Hemos creado un informe de problemas que <a href={url} target="_blank" rel="noopener">puedes ver aquí</a>.  Eso le permite mantenerse actualizado y aun así comunicarse con nosotros al respecto.

Hacemos todo lo posible para añadir su solicitud lo antes posible, pero puede tardar varios días. De hecho, para garantizar una alta calidad de los datos comprobamos cada contribución para asegurarnos de que todo funciona perfectamente.

Si quieres ser incorporado a la lista de colaboradores que se publica en la página web OTA, simplemente envíanos un <a href="mailto:contact@opentermsarchive.org">correo electrónico</a> o un mensaje directo en nuestra <a href="https://twitter.com/OpenTerms" rel="noopener">cuenta de Twitter</a>.
