## Ahora eres un colaborador del proyecto Open Terms Archive

Se utiliza el administrador de propuestas en GitHub para gestionar las contribuciones fácilmente.  Hemos creado un informe de problemas que <a href={url} target="_blank" rel="noopener">puedes ver aquí</a>.  Eso le permite mantenerse actualizado y aun así comunicarse con nosotros al respecto.

Hacemos todo lo posible para añadir su solicitud lo antes posible, pero puede tardar varios días. De hecho, para garantizar una alta calidad de los datos probamos cada contribución para asegurarnos de que todo funciona perfectamente.

If you would like to be added to the list of contributors posted on the OTA website, simply send us an <a href="mailto:contact@opentermsarchive.org">email</a> or a direct message on our <a href="https://twitter.com/OpenTerms" rel="noopener">Twitter</a> account.
